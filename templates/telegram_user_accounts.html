添加 Telegram 用户账号

使用 Telegram 用户账号（不是 Bot）来获取你加入的所有群组，然后可以将 Bot 邀请到这些群组。

如何获取 API ID 和 API Hash？

1. 访问 https://my.telegram.org 并登录
2. 点击 "API Development tools"
3. 填写应用名称和描述，创建应用
4. 复制 api_id 和 api_hash

注意：请确保已添加 Bot 账号（用于发送营销消息），然后在"Bot 用户名"中填写 Bot 的用户名。
